# Calories Burnt Prediction

This project predicts the calories burned based on different activities using a linear regression model.

## Getting Started

### Prerequisites

- Python 3.x
- Pip

### Installation

1.Install the required packages:
pip install -r requirements.txt
2.Running the Model
To run the model, execute:
python src/model.py
